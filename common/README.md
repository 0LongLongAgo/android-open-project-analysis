### 1. 功能介绍

### 2. 详细设计
包括核心类功能介绍，类关系图

### 4. 总体设计

- 可使用 Visio 等工具完成，其他工具推荐？？  
- 非所有项目必须，不需要的请先在群里反馈  

### 5. 杂谈

### 6. 修改完善
